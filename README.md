# Algorithms and Structures in Python

This repository contains implementations of various algorithms and data structures in Python.

## Dependencies

This project uses the following libraries for testing and development:

*   **pytest**: A framework that makes building simple and scalable tests easy.
*   **pytest-html**: A plugin for pytest to generate HTML reports for test results.
*   **pytest-cov**: A plugin for pytest to measure code coverage.
*   **pytest-metadata**: A plugin for pytest for accessing test session metadata.
